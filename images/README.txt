Placez ici :
- logo.png : le logo du club (idéalement carré, format PNG avec transparence)
- produits/ : les images des produits (JPG ou PNG, carré 600x600 px recommandé)

Les noms des images de produits doivent correspondre à ceux déclarés
dans config/products.yaml (champ "images").

Si une image est absente, la boutique affiche automatiquement un placeholder.
